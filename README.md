## Key Properties of Algorithms

Every solution here respects the fundamental properties taught in the course:

- **Definiteness** – Clear, unambiguous steps (no division by zero, etc.)
- **Finiteness** – Terminates in finite steps (no infinite loops)
- **Effectiveness** – Uses practical, feasible operations
